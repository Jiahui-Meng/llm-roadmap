# How to Use This Repo

## Recommended Routine

For each study day:
1. Open `docs/daily-plan.md`
2. Copy `docs/daily-template.md` into `notes/`
3. Read only the assigned materials
4. Implement the assigned code or experiment
5. Write one short note connecting theory to engineering
6. Update `docs/progress-tracker.md`
7. Commit your work

## Commit Pattern
Suggested commit message format:
- `day-01: why transformer notes`
- `day-04: implement scaled dot-product attention`
- `week-02: add transformer block and masks`

## Definition of Done for a Study Day
A day counts as done only if all are true:
- reading completed
- at least one artifact produced
- one note written
- progress tracker updated
- changes committed

## Definition of Done for a Week
- weekly review written
- code cleaned up
- notes organized
- one engineering bridge captured
- one short verbal explanation rehearsed
