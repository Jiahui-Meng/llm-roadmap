# RAG Prototype v1\n\n## Goal\nBuild a minimal end-to-end RAG pipeline covering ingestion, indexing, retrieval, prompt assembly, and grounded generation.\n\n## Pipeline\n1. ingest source documents\n2. chunk text\n3. build index\n4. retrieve top-k chunks\n5. assemble prompt\n6. generate answer\n\n## Scope\nThis v1 prototype focuses on correctness of the pipeline shape, not production-grade optimization.\n\n## Future upgrades\n- hybrid retrieval\n- reranking\n- citation grounding\n- evaluation set\n
