# RAG Prototype v1

## Goal
Build the smallest useful retrieval-augmented generation prototype.

## Minimum Features
- small corpus ingestion
- embeddings + indexing
- top-k retrieval
- prompt assembly with retrieved context
- answer generation
- basic source display

## Suggested First Data Sources
- Hugging Face model cards
- Hugging Face dataset cards
- selected technical blogs
- arXiv abstracts

## Next Steps
- add reranking
- add citations
- add evaluation set
- connect to self-hosted model serving
