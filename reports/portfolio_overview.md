# Portfolio Overview

## About This Repository
This is a 12-week LLM engineering learning portfolio covering the full stack from Transformer fundamentals to production-ready systems.

## What's Inside

### Phase 1 — Foundations (Week 1-4)
- Transformer architecture from first principles
- Embedding, attention, FFN, residual connections, masks
- Mini Transformer implementation
- Modern LLM block (RoPE, RMSNorm, SwiGLU)
- KV cache, FlashAttention, GQA/MQA
- MoE basics

### Phase 2 — Core LLM Engineering (Week 5-8)
- RAG: architecture, ingestion, chunking, hybrid retrieval, reranking, citation grounding
- RAG evaluation and ablation
- vLLM serving, benchmarking, quantization (AWQ, GPTQ, GGUF)
- LoRA / QLoRA fine-tuning pipeline

### Phase 3 — System Design (Week 9-12)
- SGLang and structured generation
- Agent workflows with tool calling
- Open Research Copilot
- Health Agent 2.0 (flagship project)
- Portfolio packaging

## Key Projects
| Project | Description |
|---------|-------------|
| Mini Transformer | From-scratch Transformer implementation |
| RAG Prototype v1/v2 | End-to-end RAG with hybrid retrieval and evaluation |
| Structured Generation Lab | Schema-constrained LLM output |
| Open Research Copilot | Multi-step research agent with citations |
| Health Agent 2.0 | Flagship: hybrid retrieval + graph + timeline + evidence grounding |
